## Basic usage

The minimum required configuration: just a username and a container.

<CodePenExample
  title="Basic usage"
  js={`createRepoWidget({
  username: 'peterbenoit',
  containerId: 'repo-container',
});`}
>
  ```js
  createRepoWidget({
    username: 'peterbenoit',
    containerId: 'repo-container',
  });
  ```
</CodePenExample>

## Custom layout

Show more repos in a wider grid on larger screens.

<CodePenExample
  title="Custom layout"
  js={`createRepoWidget({

  columns: {
    mobile: 1,
    tablet: 3,
    desktop: 4,

## Filtered and sorted

Display only the six most recently updated repos, hiding a couple of internal projects.

<CodePenExample
  title="Filtered and sorted"
  js={`createRepoWidget({
  username: 'peterbenoit',
  containerId: 'repo-container',
  sortBy: 'updated',
  maxRepos: 6,
  exclude: ['dotfiles', 'personal-notes'],
});`}
>
  ```js
  createRepoWidget({
    username: 'peterbenoit',
    containerId: 'repo-container',
    sortBy: 'updated',
    maxRepos: 6,
    exclude: ['dotfiles', 'personal-notes'],
  });
  ```
</CodePenExample>

## Portfolio spotlight

Top five most-starred repos in a single centered column, with a pronounced hover lift.

<CodePenExample
  title="Portfolio spotlight"
  js={`createRepoWidget({
  username: 'peterbenoit',
  containerId: 'repo-container',
  sortBy: 'stars',
  maxRepos: 5,
  scaleOnHover: 1.08,
  columns: { mobile: 1, tablet: 1, desktop: 1 },
  cardStyles: {
    backgroundColor: '#f6f8fa',
    borderRadius: '12px',
    boxShadow: '0 10px 24px rgba(0,0,0,0.06)',
    border: '1px solid #d0d7de',
    padding: '1.5rem',
  },
});`}
>
  ```js
  createRepoWidget({
    username: 'peterbenoit',
    containerId: 'repo-container',
    sortBy: 'stars',
    maxRepos: 5,
    scaleOnHover: 1.08,
    columns: { mobile: 1, tablet: 1, desktop: 1 },
  ```

## ES module import

If you're using a bundler (Webpack, Vite, Rollup), import the named export instead of using the global script.

<Tabs>
  <TabItem label="JavaScript">
    ```js
    import { createRepoWidget } from 'repo-widget';

    createRepoWidget({
      username: 'peterbenoit',
      containerId: 'repo-container',
    });
    ```
  </TabItem>
  <TabItem label="TypeScript">
    ```ts
    import { createRepoWidget } from 'repo-widget';

    createRepoWidget({
      username: 'peterbenoit',
      containerId: 'repo-container',
      sortBy: 'stars',
      maxRepos: 6,
    });
    ```
  </TabItem>
</Tabs>
